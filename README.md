# Oficina Maralex | Site Institucional

Site desenvolvido para a **Oficina Maralex**, utilizando **HTML, CSS e JavaScript** com design responsivo para desktops, tablets e smartphones.

## Funcionalidades

- Seções de apresentação: Home, Sobre, Serviços, Depoimentos e Contato
- Design moderno e responsivo
- Botão flutuante do WhatsApp
- Animações de scroll (reveal)
- Integração com Google Maps
- Contato com telefone e WhatsApp

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Font Awesome (ícones)
- Google Fonts (Inter e Rajdhani)

## Como visualizar localmente

git clone https://github.com/<usuario>/oficina-maralex.git
